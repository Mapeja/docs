# Frequently Asked Questions

This page provides answers to the most common questions about Esker, helping you get the most out of your writing experience.

## General Questions

<AccordionGroup>
  <Accordion title="What is Esker?" icon="pen-to-square">
    Esker is a distraction-free writing application designed specifically for writers. It focuses on providing a clean writing environment while offering powerful organization features in a single file approach. Unlike traditional word processors, Esker is built around a writer's workflow, helping you stay in the flow while organizing your thoughts.
  </Accordion>

  <Accordion title="Is Esker free to use?" icon="dollar-sign">
    The Esker Beta is currently free for all users. Once Esker leaves Beta, there will be both free and premium tiers. The core writing and organization features will remain free, while certain advanced features will be available in the premium version.
  </Accordion>

  <Accordion title="Which platforms does Esker support?" icon="desktop">
    Esker is available as:
    - A desktop application for Windows and macOS
    - A web application accessible through modern browsers

    A mobile version is planned for future releases.
  </Accordion>

  <Accordion title="How do I get started with Esker?" icon="rocket">
    To get started:
    1. [Install Esker](/getting-started/installing) on your preferred platform
    2. [Create your first project](/getting-started/first-project)
    3. Explore the [Editor Basics](/writing/editor-basics) to learn how to use the interface
    4. [Review the Writing Format](/writing/writing-format) to understand how to structure your document
  </Accordion>

  <Accordion title="How do I report bugs or request features?" icon="bug">
    You can report bugs or suggest new features by:
    - Clicking the "ESKER BETA Report Issue or Feedback" button in the application
    - Joining the [Esker Discord community](https://discord.gg/j3g9vmqB)
    - Visiting the [Feature Voting Board](https://feedback.esker.app) to request or upvote features
    - Emailing support directly at support@esker.app
  </Accordion>
</AccordionGroup>

## Files and Formats

<AccordionGroup>
  <Accordion title="What file format does Esker use?" icon="file">
    Esker uses its own `.esker` file format, which is a plain text format that contains all your writing and project metadata. This format is designed to be:
    - Portable across devices
    - Future-proof (readable by any text editor)
    - Comprehensive (stores all project information in one file)
  </Accordion>

  <Accordion title="Can I open Esker files in other applications?" icon="arrows-up-down-left-right">
    Yes, `.esker` files are plain text files that can be opened in any text editor. While specialized formatting and structure will be visible as plain text markers (like # for headings), the content remains fully accessible even without Esker.
  </Accordion>

  <Accordion title="How do I export my work from Esker?" icon="file-export">
    Esker supports exporting to several formats:
    1. Click the menu button (☰) and select **Export...**
    2. Choose your desired export type (Full Draft, Manuscript, etc.)
    3. Select your preferred file format (TXT, MD, DOCX, RTF, or PDF)
    4. Configure any additional export options
    5. Click "Export Project"

    For more details, see [Exporting Formats](/exporting/exporting-formats).
  </Accordion>

  <Accordion title="Can I import documents from other writing applications?" icon="file-import">
    While Esker doesn't have a direct import feature for other writing application formats, you can:
    1. Copy and paste text from other applications into Esker
    2. Format the pasted content using [Esker's formatting syntax](/writing/writing-format)
    3. Save the new document as an `.esker` file
  </Accordion>
</AccordionGroup>

## Writing and Editing

<AccordionGroup>
  <Accordion title="How do I create chapters and scenes?" icon="book">
    Esker uses a markdown-like syntax for document structure:
    - Type `# Title` for main headings (often used for chapters)
    - Type `## Title` for secondary headings (often used for scenes)
    - Type `### Title` for tertiary headings (often used for subscenes)

    These headings automatically appear in the sidebar for easy navigation.
  </Accordion>

  <Accordion title="How do I add notes to my manuscript?" icon="note-sticky">
    To add a note:
    1. Start a new line
    2. Type `@ ` followed by your note text
    3. Press Enter to complete the note

    Notes appear in a dedicated "Notes" section in the sidebar and can be used for research, reminders, or comments that aren't part of your main text.
  </Accordion>

  <Accordion title="Can I organize my document structure?" icon="folder-tree">
    Yes, Esker provides multiple ways to organize your document:
    - Use the sidebar to navigate between sections
    - Enable [Outline Mode](/organization/outline-mode) for a bird's eye view of your structure
    - Use drag and drop to reorder sections in both the sidebar and Outline Mode
  </Accordion>

  <Accordion title="What are synopses and how do I use them?" icon="align-left">
    Synopses are brief summaries of sections in your document:
    1. After a heading, start a new line
    2. Type `= ` followed by your synopsis text
    3. Press Enter to complete the synopsis

    Synopses appear in the sidebar under their associated heading and are useful for planning or summarizing your content.
  </Accordion>
</AccordionGroup>

## Features and Functionality

<AccordionGroup>
  <Accordion title="What is Outline Mode?" icon="list-check">
    Outline Mode is a specialized view that shows only your document's structure (headings, synopses, notes) while hiding the main body text. This gives you a bird's eye view of your document and allows for easy reorganization.

    To use Outline Mode, select "Enable Outline Mode" from the menu.
  </Accordion>

  <Accordion title="What are Writing Sprints?" icon="person-running">
    Writing Sprints are timed writing sessions designed to boost productivity:
    1. Click the "Writing Sprint" button in the statistics bar
    2. Choose between a time-based sprint or word-count target
    3. Start your sprint and focus solely on writing
    4. When the sprint ends, Esker displays your writing statistics

    Sprints help overcome writer's block and establish productive writing habits.
  </Accordion>

  <Accordion title="How do Session Trackers work?" icon="stopwatch">
    Sessions track your writing activity over time:
    1. Click "Start Session" in the statistics bar
    2. Write as normal while Esker tracks your time and word count
    3. Pause or reset the session as needed
    4. View your words per minute and total writing time

    Sessions help you understand your writing patterns and productivity.
  </Accordion>

  <Accordion title="What is Focus Mode?" icon="eye">
    Focus Mode helps concentrate on your current paragraph by dimming the rest of your document. Enable it by selecting "Enable Focus Mode" from the menu.
  </Accordion>

  <Accordion title="What is Zero Distractions Mode?" icon="minimize">
    Zero Distractions Mode combines multiple focus features:
    - Hides the sidebar and statistics bar
    - Enables Focus Mode
    - Enables Typewriter Mode
    - Activates fullscreen

    This creates the most distraction-free environment possible. Enable it by selecting "Enable Zero Distractions Mode" from the menu.
  </Accordion>
</AccordionGroup>

## Customization

<AccordionGroup>
  <Accordion title="Can I change the theme?" icon="palette">
    Yes, Esker offers several visual themes:
    1. Click the menu button (☰) and select **Options**
    2. In the Theme section, choose from available options (Light, Dark, Vintage Green, etc.)
    3. Click "Save Options" to apply

    For more details, see [Themes](/customization/themes).
  </Accordion>

  <Accordion title="Can I customize the editor width?" icon="arrows-left-right">
    Yes, you can adjust how wide the text editor appears:
    1. Click the menu button (☰) and select **Options**
    2. In the Editor section, enter your desired width (percentage, pixels, or characters)
    3. Click "Save Options" to apply

    For more options, see [Layout Options](/customization/layout-options).
  </Accordion>

  <Accordion title="Can I adjust the font size?" icon="text-height">
    Yes, Esker allows font size customization:
    1. Click the menu button (☰) and select **Options**
    2. In the Typography section, choose your preferred font size
    3. Click "Save Options" to apply
  </Accordion>

  <Accordion title="What are typing sounds?" icon="volume-high">
    Typing sounds add audio feedback to your keystrokes, creating a more immersive writing experience:
    1. Click the menu button (☰) and select **Options**
    2. In the Typing Sound section, choose your preferred sound profile
    3. Click "Save Options" to apply

    Options include various mechanical keyboard sounds and typewriter effects.
  </Accordion>
</AccordionGroup>

## Technical Questions

<AccordionGroup>
  <Accordion title="Does Esker support automatic backups?" icon="clock-rotate-left">
    Yes, Esker offers several backup options:
    1. **Autosave**: Automatically saves your work at regular intervals
    2. **Backup on Close**: Creates a backup copy whenever you close a project

    Both features can be configured in the Options dialog. For more details, see [Autosave & Backups](/exporting/autosave-backups).
  </Accordion>

  <Accordion title="What happens if my computer crashes?" icon="explosion">
    If Esker or your computer crashes unexpectedly:
    1. Esker attempts to recover your work when you reopen the application
    2. If you had autosave enabled, you should lose minimal work
    3. Check the backup folder for recent copies of your project

    For best protection, enable autosave with a short interval (3-5 minutes).
  </Accordion>

  <Accordion title="Can I use Esker offline?" icon="wifi">
    The desktop version of Esker works completely offline. The web version requires an internet connection for initial access, but future offline support is planned.
  </Accordion>

  <Accordion title="How large a document can Esker handle?" icon="book-atlas">
    Esker is optimized for novels and book-length documents, easily handling projects up to several hundred thousand words. For extremely large projects, consider dividing them into multiple `.esker` files for optimal performance.
  </Accordion>

  <Accordion title="Will there be collaboration features?" icon="users">
    Collaborative writing features are planned for future versions of Esker. The current focus is on perfecting the individual writing experience.
  </Accordion>
</AccordionGroup>

## Professional Writing

<AccordionGroup>
  <Accordion title="Is Esker suitable for professional writers?" icon="briefcase">
    Yes, Esker is designed with professional writers in mind:
    - The distraction-free interface supports sustained, focused writing
    - Organizational features help manage complex projects
    - The Manuscript Compiler creates industry-standard submissions
    - The single-file approach simplifies backup and version management
    - Features like Writing Sprints and Sessions help maintain productivity

    Many professional authors are already using Esker for their work.
  </Accordion>

  <Accordion title="Can I submit manuscripts directly from Esker?" icon="paper-plane">
    While Esker doesn't have direct submission functionality, you can:
    1. Use the [Manuscript Compiler](/exporting/manuscript-compiler) to create a professionally formatted PDF
    2. Export to standard formats like DOCX for submission platforms
    3. Use the compiled files in your submission process

    The Manuscript Compiler specifically creates documents that meet industry standards for submission.
  </Accordion>

  <Accordion title="Can I track my writing goals?" icon="bullseye">
    Yes, Esker provides several goal-tracking features:
    - Set a word count goal for your project in Project Settings
    - Add an optional deadline to calculate daily writing targets
    - Use the statistics bar to track progress toward your goal
    - Employ Writing Sprints to set and achieve short-term goals

    For more details, see [Word Count](/productivity/word-count).
  </Accordion>

  <Accordion title="Does Esker support book front matter and back matter?" icon="book-open">
    Yes, the Manuscript Compiler includes options for standard front and back matter elements:
    - Dedication
    - Preface
    - Introduction
    - Prologue
    - Epilogue
    - Afterword
    - Acknowledgments

    These can be included when creating professionally formatted manuscripts.
  </Accordion>
</AccordionGroup>
