# ⌨️ Using the Esker Editor

The **Esker Editor** is designed to keep you in the flow while writing. With **intuitive formatting, focus modes, and keyboard shortcuts**, Esker makes writing effortless.

## 🖊️ **Navigating the Editor**

When you open Esker, the editor takes center stage. Here’s how to **navigate and interact** with it:

- **Click anywhere** to start writing.
- Use **Arrow Keys** or **Page Up/Page Down** to move around.
- The **Sidebar** updates dynamically as you write.
- **Click a section in the Sidebar** to jump to that part of your manuscript.

💡 **Tip:** The editor is fully keyboard-driven, allowing you to work **without a mouse**.

## ✍️ **Writing & Formatting Text**

Esker supports a **lightweight syntax** for structuring your work:

```md
# Chapter One
## Scene One
### A Critical Moment
= A brief synopsis of what happens here.
@ This is a research note.
! Remember to revisit this section.
```

🔹 **Headings** appear in the Sidebar for easy navigation.  
🔹 **Synopses, notes, and bookmarks** help structure your manuscript.

## 🎯 **Focus & Typewriter Modes**

Esker offers **special modes** to enhance focus:

### 🕶️ **Focus Mode**
- **Highlights only the active section**, dimming everything else.
- Keeps distractions to a minimum.
- Enable via **☰ Menu → Enable Focus Mode** or use **F10**.

### 📝 **Typewriter Mode**
- Keeps the active line **centered on the screen**.
- Simulates the **feel of a typewriter** for immersive writing.
- Enable via **☰ Menu → Enable Typewriter Mode** or use **F9**.

💡 **Tip:** Combine **Focus Mode + Typewriter Mode** for an ultra-immersive experience!

## ⌨️ **Keyboard Shortcuts**

Speed up your workflow with **essential shortcuts**:

| Action | Shortcut |
|--------|----------|
| Save | **Ctrl + S** (Windows/Linux) or **Cmd + S** (Mac) |
| Toggle Focus Mode | **F10** |
| Toggle Typewriter Mode | **F9** |
| Open Sidebar | **Ctrl + B** (Windows/Linux) or **Cmd + B** (Mac) |
| New Line Without Formatting | **Shift + Enter** |
| Move Between Sections | **Ctrl + ↑ / ↓** |

💡 **Tip:** Mastering shortcuts can **significantly boost your writing speed**!
