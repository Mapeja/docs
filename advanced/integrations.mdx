# 🔌 Integrating Esker with Other Tools  

Esker works **alongside other writing and productivity tools** to improve your workflow.

## 📜 **Exporting to Other Apps**  

You can export your Esker projects for use in other writing platforms:  

| **App** | **Recommended Format** |
|---------|------------------------|
| **Obsidian** | Markdown (`.md`) |
| **Scrivener** | Markdown or Plain Text |
| **Google Docs** | Paste Markdown into Google Docs |
| **WordPress** | Direct Markdown import |

💡 **Tip:** Esker **retains heading structure (`#`, `##`, `###`)**, making it easy to migrate content.

## 🔗 **Syncing with Cloud Services**  

You can manually **store Esker projects in cloud services** like:  

☁️ **Google Drive, Dropbox, OneDrive** – Store `.esker` files for access on multiple devices.  
💾 **GitHub** – Use Esker for version control and collaboration.  

### **How to Sync Esker Projects with a Cloud Folder**  
1. Move your Esker project folder to **Google Drive, Dropbox, or iCloud**.  
2. Open the project in Esker from that location.  
3. Your files will be **automatically backed up in the cloud**.
