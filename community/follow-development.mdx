# 📢 Follow Esker’s Development  

Want to stay up to date with **new features, bug fixes, and upcoming updates**? Follow Esker’s development through these platforms:  

## 🚀 **Official Development Channels**  

📜 **[Esker Changelog](/changelog/latest-features)** – See what’s new in each update.  
📅 **[Esker Roadmap](/changelog/upcoming-features)** – Track upcoming features and vote for priorities.  
🐞 **[GitHub Issues](https://github.com/eskerapp/issues)** – Follow bug fixes and feature discussions.  
💬 **[Esker Discord](https://discord.gg/eskerapp)** – Get updates from the Esker team.  

### 📢 **Want to Contribute to Esker?**  
If you have ideas for improvements, you can:  
🔹 Submit feature requests via **[GitHub Discussions](https://github.com/eskerapp/discussions)**.  
🔹 Report issues in **[#bug-reports](https://discord.gg/eskerapp)** on Discord.  
🔹 Vote for upcoming features in the **[Esker Roadmap](/changelog/upcoming-features)**.  

💡 **Tip:** Enable notifications in the **#announcements** channel on Discord to get updates first!
