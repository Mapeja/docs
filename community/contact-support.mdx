# 📩 Contact Esker Support  

If you need assistance, Esker offers multiple ways to get support.  

## 🛠️ **Self-Help Resources**  

Before reaching out, check these resources:  

- **[Troubleshooting Guide](/troubleshooting/common-issues)** – Fix common issues.  
- **[FAQs](/troubleshooting/faqs)** – Find quick answers.  
- **[Bug Reports & Feature Requests](/troubleshooting/reporting-bugs)** – Request improvements.  

## 📧 **Contacting Esker Directly**  

If you still need help, reach out via:  

📩 **Email Support:** [support@esker.app](mailto:support@esker.app)  
🐞 **Report Bugs:** [GitHub Issues](https://github.com/eskerapp/issues)  
💬 **Discord Support:** Get community help in **[Esker Discord](https://discord.gg/eskerapp)**  

### 🕒 **Response Time**  
We aim to **respond within 24-48 hours** for email and GitHub issues.  

💡 **Tip:** For urgent issues, **Discord is the fastest way to get help**!
