# Installing Esker

Esker is currently available as a desktop application for Windows and Mac, with a web version in beta. This guide will walk you through downloading and installing Esker on your device.

## System Requirements

Before installing Esker, make sure your system meets these minimum requirements:

| Operating System | Requirements |
|------------------|--------------|
| Windows | Windows 10 or later |
| macOS | macOS 10.14 (Mojave) or later |
| Web Version | Modern browser (Chrome, Firefox, Edge, Safari) |

## Downloading Esker

<Steps>
  <Step title="Visit the Esker Website">
    Go to the Esker download page at [esker.app/download](https://esker.app/download).
  </Step>
  <Step title="Select Your Platform">
    Choose the download option for your operating system (Windows or Mac).
  </Step>
  <Step title="Start the Download">
    Click the download button to begin downloading the installer.
  </Step>
</Steps>

## Installing on Windows

<Steps>
  <Step title="Locate the Downloaded File">
    Find the Esker installer (`.exe` file) in your Downloads folder.
  </Step>
  <Step title="Run the Installer">
    Double-click the installer file to begin the installation process.
  </Step>
  <Step title="Follow Installation Prompts">
    Follow the on-screen instructions to complete installation.
    
    <Note>
      You may see a Windows security prompt. This is normal when installing new software. Click "Yes" to proceed.
    </Note>
  </Step>
  <Step title="Complete Installation">
    Once installation is complete, you can launch Esker from your Start menu or desktop shortcut.
  </Step>
</Steps>

## Installing on Mac

<Steps>
  <Step title="Locate the Downloaded File">
    Find the Esker installer (`.dmg` file) in your Downloads folder.
  </Step>
  <Step title="Open the Installer">
    Double-click the `.dmg` file to open it.
  </Step>
  <Step title="Install the Application">
    Drag the Esker icon to the Applications folder as shown in the installer window.
    
    <Note>
      If this is your first time opening Esker, macOS may display a security warning. To bypass this, go to System Preferences > Security & Privacy and click "Open Anyway."
    </Note>
  </Step>
  <Step title="Launch Esker">
    Open Esker from your Applications folder or Launchpad.
  </Step>
</Steps>

## Using the Web Version (Beta)

Esker also offers a web version that you can use in your browser without installation:

1. Visit [app.esker.app](https://app.esker.app) in a modern web browser
2. Sign in using your email address
3. Begin using Esker directly in your browser

<Warning>
  The web version stores files locally in your browser. Always export or save important projects to your device.
</Warning>

## Troubleshooting Installation

If you encounter issues during installation:

- Make sure you have administrator permissions on your device
- Temporarily disable antivirus software during installation
- Check if your operating system meets the minimum requirements
- Try downloading the installer again if the file appears corrupted

If problems persist, visit our [Common Issues](/help/common-issues) page for more troubleshooting steps.
